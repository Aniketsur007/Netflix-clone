# 🎬 Netflix Clone (HTML & CSS)

A responsive Netflix landing page clone built using **HTML and CSS**.

## 🚀 Features

* 🔥 Modern Netflix-style UI
* 📱 Responsive design (desktop, tablet, mobile)
* 🎥 Trending movies section
* ❓ FAQ section
* 📧 Email input UI
* 🧾 Footer with multiple links

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox & Grid)
